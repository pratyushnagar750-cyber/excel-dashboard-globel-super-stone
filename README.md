# excel-dashboard-globel-super-stone
excel sales dashboard project with data analysis and visualization.
Developed an interactive Excel Dashboard using the Global Superstore dataset to analyze business performance and sales trends. 
The project involved data cleaning, data organization, and visualization techniques using Microsoft Excel. 
Implemented Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and KPI analysis to track important metrics such as Sales, Profit, Orders, and Customer Segments. 
The dashboard provides insights into regional performance, top-selling categories, and customer behavior, helping in better decision-making through data visualization and reporting.
